# SmartPlate — Vision-Based Healthy Plate Scoring

CSCI B457 Computer Vision · Spring 2026 · Final Project  

---

## What the project does

SmartPlate takes a single photo of a food plate and outputs:
1. A semantic segmentation mask (which pixels are which food).
2. A 0–100 health score based on the [Harvard Healthy Eating Plate](https://www.hsph.harvard.edu/nutritionsource/healthy-eating-plate/) rubric.
3. Estimated macros (kcal, protein, carb, fat) from food-group proportions.

**Pipeline:** image → SegFormer-B0 (fine-tuned on FoodSeg103) → per-pixel class labels → aggregate to 7 food groups → score against Harvard targets.

## Running the pipeline

Run notebooks in order:

1. **`01_explore_and_prepare.ipynb`** — downloads FoodSeg103 (~500 MB via HuggingFace), visualizes samples, scaffolds the class→group CSV. The CSV is already filled in, so this notebook is mainly for reference.
2. **`02_train_segformer.ipynb`** — fine-tunes SegFormer-B0 and writes `checkpoints/segformer_best/`. Set `QUICK_SMOKE_TEST = True` for a 5-minute sanity check before the full run.
3. **`03_train_baseline.ipynb`** — trains a ResNet50 multi-label classifier as a comparison baseline.
4. **`04_demo.ipynb`** — loads the SegFormer checkpoint and runs the full pipeline on held-out plates.

## Key design decisions

- **SegFormer-B0** (pretrained `nvidia/segformer-b0-finetuned-ade-512-512`): lightweight enough to fine-tune on a laptop GPU/MPS in a few hours.
- **7 food groups** mirror the Harvard plate: vegetables, fruits, whole grains, healthy protein, refined grains, red/processed meat, sugary/fatty.
- **Score formula** in `smartplate.score_plate`: `base (positive fraction) − balance penalty (L1 from Harvard targets) − negative penalty (weighted negative fractions)`. Ideal Harvard plate → 100, all-junk → 0.
- **Macro estimation** assumes a fixed 400 g plate mass (no depth info available from a single image); treated as a rough order-of-magnitude estimate in the report.

**FoodSeg103** (Wu et al., ACM MM 2021). 7,120 images, 103 food classes + background.  
Auto-downloaded on first run via HuggingFace (`EduardoPacheco/FoodSeg103`). Do not commit dataset files.
