# SmartPlate — Code

Vision-based healthy-plate scoring from a single food image.
CSCI B457 Final Project, Group 9.

## What's in here

```
code/
├── smartplate.py                        # shared helpers (scoring, nutrition, viz)
├── notebooks/
│   ├── 01_explore_and_prepare.ipynb     # dataset EDA + build class→food-group mapping
│   ├── 02_train_segformer.ipynb         # fine-tune SegFormer-B0 on FoodSeg103
│   ├── 03_train_baseline.ipynb          # ResNet50 multi-label classifier
│   └── 04_demo.ipynb                    # end-to-end demo: image → mask → score
├── assets/
│   ├── foodseg103_to_groups.csv         # class_id,class_name,food_group (hand-built)
│   └── usda_macros.csv                  # food_group → kcal/protein/carb/fat per 100g
├── checkpoints/                         # saved model weights (regenerated by training)
├── figures/                             # PNGs used in the report
└── requirements.txt
```

## Setup

Tested on macOS with Apple Silicon (PyTorch MPS).

```bash
cd code
python -m venv .venv
source .venv/bin/activate
pip install --upgrade pip
pip install -r requirements.txt
python -m ipykernel install --user --name smartplate --display-name "SmartPlate (.venv)"
```

Then open any notebook in Jupyter / VS Code and select the **SmartPlate (.venv)** kernel.

## Reproducing results

Run notebooks in order:

1. `01_explore_and_prepare.ipynb` — downloads FoodSeg103 via HuggingFace (`EduardoPacheco/FoodSeg103`), visualizes samples, and scaffolds `assets/foodseg103_to_groups.csv`. The CSV is then filled in by hand (one-time ~45-min task).
2. `02_train_segformer.ipynb` — fine-tunes SegFormer-B0 and writes `checkpoints/segformer_b0_foodseg103.pt`.
3. `03_train_baseline.ipynb` — trains ResNet50 multi-label classifier, writes `checkpoints/resnet50_multilabel.pt`.
4. `04_demo.ipynb` — loads the SegFormer checkpoint and runs the full pipeline on held-out plates.

## Dataset

**FoodSeg103** (Wu et al., ACM MM 2021). 7,120 images, 103 food classes + background.
Auto-downloaded on first notebook run into the HuggingFace cache (`~/.cache/huggingface/`).
No need to commit dataset files — the notebooks fetch them.

## Scoring rubric

Harvard Healthy Eating Plate (https://www.hsph.harvard.edu/nutritionsource/healthy-eating-plate/).
103 classes are aggregated into 7 food groups, then scored against the rubric targets.
See `smartplate.py :: score_plate` and `assets/foodseg103_to_groups.csv`.

## Reused code / models

- HuggingFace `transformers` SegformerForSemanticSegmentation (pretrained `nvidia/segformer-b0-finetuned-ade-512-512`).
- `torchvision.models.resnet50` (IMAGENET1K_V2 weights).
- FoodSeg103 dataset (`EduardoPacheco/FoodSeg103` HF mirror).

Full citations live in `../report/main.bib`.
